# Stellar | Astronomy & Stargazing Club HTML Template

Stellar is a premium, fully responsive, and feature-rich HTML template designed for astronomy clubs, agencies, and educational organizations. Built with a mobile-first approach, it features robust dark mode support, RTL compatibility, and modern design aesthetics.

## Features
- **Modern UI/UX**: Sleek, stargazing-themed design with Outfit and Inter typography.
- **Responsive Design**: Fluid layout that works perfectly across all devices (Mobile, Tablet, Desktop).
- **Dark/Light Mode**: Automatic system preference detection and manual toggle.
- **RTL Support**: Built-in support for Right-to-Left languages.
- **SEO Optimized**: Optimized meta tags, header hierarchy, and JSON-LD structured data.
- **Service-Based Functionality**: Includes service grid, detail pages, testimonials, and a custom cost calculator.
- **Accessibility**: Follows WCAG 2.1 AA guidelines with large touch targets and semantic HTML.
- **Placeholder Integrations**: Ready-to-use sections for Netlify Forms, Google Maps, and Booking systems.

## Page Architecture
- `index.html`: Home page with hero, features, and CTA.
- `about.html`: Story, mission, and team profiles.
- `services.html`: Service listings and cost calculator.
- `service-detail.html`: Detailed service page with booking sidebar.
- `events.html`: Calendar grid and event cards with RSVP.
- `gallery.html`: Masonry grid of astrophotography.
- `blog.html`: Article listing with pagination.
- `resources.html`: Downloadable guides and checklists.
- `contact.html`: Contact form, map, and appointment booking point.
- `member-dashboard.html`: User dashboard with stats and upcoming events.
- `admin-dashboard.html`: Admin panel for member and content management.
- `user-profile.html`: User profile settings form.
- `404.html`: Custom error page.
- `coming-soon.html`: Pre-launch page with countdown.

## Technical Details
- **CSS**: Vanilla CSS with logical properties and a 8px-based spacing system.
- **JS**: lightweight vanilla JS for theme and layout logic.
- **Icons**: Font Awesome 6.4.
- **Fonts**: Google Fonts (Outfit & Inter).

## How to Customize
1. **Colors**: Modify the `--primary`, `--secondary`, and `--accent` variables in `assets/css/style.css`.
2. **Typography**: Change the font families in the `:root` selector.
3. **Images**: Replace Unsplash source URLs in HTML files with your own assets.

## SEO Compliance
- Metadata defined in `<head>` for all pages.
- `robots.txt` and `sitemap.xml` included.
- Header tags follow a strict H1 > H2 > H3 hierarchy.
